# Manual de Usuario - CMDB Integral

## Acceso
1. Abrir `http://localhost/CMDB_Semestral/public/`.
2. Seleccionar **Iniciar sesión**.
3. Usar una credencial de prueba indicada en el README.

## Administrador
1. Desde **Usuarios** puede crear cuentas, editarlas, darlas de baja o desbloquearlas.
2. Desde **Categorías** controla los tipos de activos.
3. Desde **Colaboradores** registra personas, ubicación, departamento y fotografía.
4. Desde **Inventario** registra equipos, software o licencias.
5. Desde **Asignaciones** vincula un activo disponible a un colaborador.
6. Desde **Necesidades** responde solicitudes creadas por los colaboradores.
7. Desde **Reportes** descarga los archivos Excel.
8. Desde **Bitácora** revisa acciones relevantes de seguridad y operación.

## Colaborador
1. Iniciar sesión con una cuenta de rol COLABORADOR.
2. Abrir **Mi portal**.
3. Consultar equipos bajo su responsabilidad.
4. Enviar una solicitud de equipo, software o licencia.
5. Revisar el estado de la solicitud.
6. Cambiar la contraseña desde el enlace correspondiente.

## Capturas que se deben agregar al informe
- Inicio de sesión correcto.
- Cuenta bloqueada por tres intentos.
- Registro de colaborador.
- Registro de equipo con fotografía.
- Detalle del activo con firma válida.
- Asignación de equipo.
- Portal de colaborador.
- Solicitud de necesidad.
- Reporte Excel.
- Bitácora.
